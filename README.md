# Threaded Mandelbrot Viewer
#### A simple threaded Mandelbrot viewer written in C++17 and rendered using GTK

#### Prerequisites
```
sudo apt install libgtkmm-3.0-dev
sudo apt libgstreamermm-1.0-dev
```

#### Build
```
make clean
make
```

#### Tested Targets
```
- NVIDIA Jetson Xavier
- Raspberry Pi 4
```
